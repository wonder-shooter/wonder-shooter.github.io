# wonder-shooter-site
wonder-shooter webサイト用リポジトリ

## 手順

1. $ npm install
2. $ gulp copy
3. $ npm start で実行

## 動作環境

- Mac(Sierra)/Windows10 での動作確認済み
- node v7.2.1 以上で動作確認済み
